
# Streamlit Insurance Policy Status App (Flat Repo)
Files stay at the repo root (no folders). Upload `Insurance.csv` via the UI or place it next to `app.py`.

## Files
- `app.py` — Streamlit app (dashboard, models, predictions)
- `requirements.txt` — minimal, **no version pins**

## Run locally
pip install -r requirements.txt
streamlit run app.py
